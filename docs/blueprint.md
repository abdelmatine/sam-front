# **App Name**: RespiraMed Solutions

## Core Features:

- Product Catalog & Details: Browse a comprehensive catalog of respiratory care devices and accessories, featuring high-resolution images, detailed specifications, and medical information.
- Intuitive Search & Filtering: Effortlessly discover products using a robust search bar, categorized browsing, and advanced filtering options for price, brand, and availability.
- Shopping Cart Management: Add, remove, and update product quantities in a dynamic shopping cart, managed with Redux Toolkit for a consistent user experience.
- Streamlined Checkout Process: A secure and intuitive multi-step checkout flow supporting shipping, billing, and order summary for a seamless purchasing experience.
- Medical Content Explainer Tool: An AI-powered tool to summarize complex medical device specifications and usage instructions, ensuring clarity and improving user comprehension of product information.
- Trust-Building Informational Pages: Dedicated About Us, Contact, and FAQ pages, providing company credentials, mission, and certifications to foster user trust and medical credibility.
- User Wishlist: Allow users to save products to a personal wishlist for future reference or purchase, managed via Redux for persistent state.

## Style Guidelines:

- Default dark background color: #282C2E (HSL: 200, 7%, 17%) to provide a modern, professional backdrop consistent with the provided palette.
- Primary interactive color: #2F8771 (HSL: 167, 51%, 35%) for prominent UI elements, CTA buttons, and key highlights, providing a clean and credible medical feel.
- Accent color: #81A59C (HSL: 167, 12%, 61%) used for subtle details, secondary accents, and to provide visual variation within the green hue spectrum.
- Neutral element color: White #FFFFFF for text on dark backgrounds and card surfaces, ensuring high readability and a clean aesthetic.
- Headline font: 'Poppins' (sans-serif) for its precise, contemporary, and geometric appearance, enhancing the modern tech aesthetic.
- Body font: 'Inter' (sans-serif) for its objective, neutral, and highly readable qualities, suitable for detailed product descriptions and accessible content.
- Emphasis on large, readable typography with high accessibility contrast and spacious line heights throughout the interface.
- Minimal medical-tech aesthetic using clean, modern iconography for product categories and UI actions, enhancing professionalism without clutter.
- Mobile-first responsive design, featuring clean spacing, rounded-xl cards, and smooth shadows to convey a premium and polished look.
- White-colored cards utilized effectively on dark backgrounds to create visual hierarchy and maintain a fresh, medical clean appearance.
- Incorporation of a sticky navigation bar with a mega menu for categories on desktop and a drawer menu for mobile, ensuring easy navigation.
- Subtle professional Framer Motion animations, including fade-up on scroll, staggered product card entrances, and smooth hover scaling effects.
- Elegant hover animations for interactive elements and smooth image transitions within product galleries to enhance user engagement without distraction.
